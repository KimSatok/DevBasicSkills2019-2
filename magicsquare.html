Magic Squares

Magic Square 3x3
Rules
Used number from 1-9 to fill in the tables. You can use each number once
The purpose is to make the sum of all row and all column equal to one number.
Magic Square 4x4
Rules
Like 3x3 but a bit difference, instead 1-9 you can use 1 to 16.
The purpose is to make the sum of all row and all column equal to one number as well.
