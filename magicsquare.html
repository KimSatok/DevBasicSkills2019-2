Magic Squares

Magic Square 3x3
Rules
Fill the square with numbers 1 through 9.
You can use each number only once.
Each row, column and diagonal line must add up to 15.
Good luck!
Magic Square 4x4
Rules
Fill the square with numbers 1 through 16.
You can use each number only once.
Each row, column and diagonal line must add up to 34.
Good luck!
